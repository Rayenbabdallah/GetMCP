# Code of Conduct

## TL;DR

Be respectful. Assume good faith. Disagreements are fine; harassment isn't.

## Standards

We expect everyone interacting with the GetMCP repository — issues, pull requests, Discussions, code, docs — to:

- **Use welcoming and inclusive language.**
- **Respect differing viewpoints and experiences.** Disagree with the idea, not the person.
- **Accept constructive criticism gracefully**, and offer it the same way.
- **Focus on what is best for the project and the community.**

The following are not acceptable:

- Personal attacks, insults, or derogatory comments.
- Trolling, sustained disruption, or off-topic flame wars.
- Public or private harassment, including unwelcome sexual attention.
- Publishing others' private information without explicit permission.

## Enforcement

If you experience or witness unacceptable behavior, report it privately to **rayenbenabdallah88@gmail.com**. Reports are confidential.

Maintainers may remove comments, commits, code, issues, or PRs that violate this Code of Conduct, and may temporarily or permanently ban contributors whose behavior is deemed inappropriate, threatening, or harmful.
